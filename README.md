# Archivos 3

Repositorio académico dedicado a ejercicios de **manejo de archivos** y persistencia de datos en programación.

## Descripción

Este proyecto forma parte de prácticas universitarias orientadas a reforzar la lectura, escritura, modificación y organización de información mediante archivos externos.

## Temas trabajados

- Lectura de archivos.
- Escritura de archivos.
- Actualización de información.
- Organización de datos en texto plano.
- Prácticas de persistencia básica.

## Objetivo académico

Aplicar técnicas de manejo de archivos para desarrollar programas capaces de conservar información más allá de la ejecución del programa.

## Tecnologías

| Área | Herramientas |
|---|---|
| Lenguaje | C++ |
| Librerías | fstream |
| Control de versiones | Git · GitHub |
